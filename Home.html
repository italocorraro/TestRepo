• <head>
  • <head> incluso in <html> prima del <body>; non è visualizzato nella pagina; contiene informazioni utili al browser:
  • <title>: include il titolo visualizzato sul browser
  • <base>: include il percorso base da usare per gli URL relativi nell'attributo href (non va chiuso)
  • <style>: include il foglio di stile CSS locale
  • <script>: include lo script locale del foglio; con l'attributo src si può importare una libreria esterna
  • <link>: definisce una relazione con un documento esterno; tipicamente usato per importare file esterni di stile
  • <meta>: include i metadati del documento (non va chiuso)
    • usato per specificare il set di caratteri da usare
    • per specificare il viewport (in modo da avere una corretta visualizzazione su mobile)
    • per indicare l'autore e altri dati generici sul documento
    • per indicare le keyword del documento usate in particolare dai motori di ricerca

    • <meta charset="UTF-8">
      <meta name="description" content="Free Web tutorials">
      <meta name="keywords" content="HTML, CSS, JavaScript">
      <meta name="author" content="John Doe"> <name="viewport" content="width=device-width, initial-scale=1">
  • Elementi "block"
  • Elementi come <div> e <p> sono elementi block; di default questi elementi iniziano sempre un nuovo rigo e occupano tutta la larghezza a disposizione (fondamentalmente sono dei blocchi)
  • <p> inoltre inizia e finisce lasciando una riga vuota
  • <div> non possiede nessuna formattazione di default (all'infuori di essere elemento "block") ed è per questo comunemente utilizzato come contenitore per altri elementi;
    utilizzato in particolare per contenere i blocchi logici delle pagine strutturate con float/clear, flexbox o grid
  • Elementi "in-line"
  • Elementi come <span> non iniziano una nuova riga e occupano tutta la larghezza disponibile
  • Entità
  • Le entity sono precedute da & e chiuse da ; e servono a rappresentare caratteri speciali inclusi < e > che normalmente sarebbero esclusivi di html
  • < è rappresentato con <
  • > è rappresentato con >
  • Elementi Fondamentali del <body>
  • <a></a> con attributo href per il link
  • <img> con attributo src per il link dell'immagine
  • Nesting Selector Esempio
    • .list-1 ul { list-style-type: square; & > li.circlelist { list-style-type: disc; &:hover { background-color: aqua; } } }
  • Modifica rapida alla lista delle classi di un elemento
  • classList.toggle("classe") aggiunge/rimuove una classe dall'elemento
  • classList.add("classe") aggiunge una classe
  • classList.remove("classe") rimuove una classe
  • classList.replace("classeSostituita", "classeSostituto") sostituisce una classe con un'altra
  • si possono modificare più classi insieme separandole con uno spazio;
    ciascun comando restituisce undefined tranne replace che restituisce true/false in base alla riuscita dell'operazione
